1. Download and install MPS: https://www.jetbrains.com/mps/download
2. Download and install 3 MPS plugins: DcareForMPS, CDM, cdm-generator
3. Restart MPS
4. Download the Zip of this repository and unpack it
5. Open the unzipped folder cdm-demo as a project in MPS
6. Show the Dclare tool window: View > Tool Windows > Status
7. In the tool window, turn on ONLY: engine, DEBUGGER
